# Редакционный стандарт карточек персонажей GenshinTop

Материалы в [`info/characters/`](../info/characters/) — **справочные профили**, а не копипаст игрового кодекса. Цель: связная структура, ссылки на опорные гайды в **`info/guides/`** и отсутствие «лесенки» из одной фразы на строку.

## Тон и границы

- Нейтральный стиль, **без кликбейта** и без выкладки полных описаний талантов из клиента — игрок смотрит множители и тексты в игре.
- **HoYoverse**, **Genshin Impact**; актуальность механик проверяйте по патчу в **`gameVersion`** и **`reviewedAt`**.

## Frontmatter (обязательное и связи)

Минимум для сайта (см. [`ContentRepository`](../lib/ContentRepository.php)):

- **`name`** — короткое имя героя или заголовок заметки.
- **`title`** — заголовок страницы (SEO, H1 дублируется шаблоном при необходимости).
- **`element`** — одна из: `Pyro`, `Hydro`, `Electro`, `Cryo`, `Anemo`, `Geo`, `Dendro` (для статей-обзоров допустимо нейтральное значение как в шаблоне).
- **`weapon`** — тип оружия по игре (русские подписи как в текущем корпусе: «Одноручное», «Двуручное», «Лук», …).
- **`sourceSlug`** — slug URL (`имя файла` без `.md`).
- **`relatedWeapons`**, **`relatedArtifacts`**, **`relatedGuides`** — YAML-списки slug’ов; в **`relatedGuides`** только существующие опорные статьи из **`info/guides/`**.

Рекомендуется также:

- **`summary`** — один-два предложения для сниппета (до ~300 символов).
- **`rating`** — опциональная буква-ориентир для карточки (не «истина»).
- **`rarity`** — `4` или `5`, если нужно для фильтров каталога.
- **`gameVersion`**, **`reviewedAt`** — актуальность.

## Структура тела (профиль персонажа)

Используйте только **`##` / `###`**, абзацы и списки по делу:

1. **Кратко о персонаже** — стихия, оружие, чем интересен без лора на десять страниц.
2. **Роль в отряде** — DD / саппорт / гибрид в терминах [ролей и отрядов](/guides/otryady-roli-elementy).
3. **Таланты: порядок прокачки** — приоритет навыков **без** полного текста из игры; ссылка на [книги и короны](/guides/talanty-knigi-i-korony), [энергию](/guides/energiya-vosstanovlenie-vzryva).
4. **Артефакты и характеристики** — общая логика + ссылки на [артефакты](/guides/artefakty-farm-i-vybor), [ЭМ](/guides/elementarnyy-masterstvo-kogda-kachat), [криты](/guides/krit-shans-i-krit-uron-baza).
5. **Оружие** — кратко + [возвышение оружия](/guides/oruzhie-vozvyshenie-materialy).
6. **Отряды и синергия** — [реакции](/guides/stihii-i-reaktsii-baza), при необходимости [бездна](/guides/vitaya-bezdna-vvedenie).
7. **Созвёздия** — [созвёздия и молитвы](/guides/sozvezdiya-i-krutki-investicii).
8. **Кому подойдёт** — честный абзац без давления «must pull».

## Вспомогательные страницы (сравнения, «кто крутить»)

Отдельный короткий формат: о чём страница, куда углубиться ([молитвы](/guides/bannery-sobytiya-molitvy), [примогемы](/guides/primogemy-kopim-tratim), карточки `/characters/...`). Без тиражирования устаревших новостных вставок.

## Техническая пересборка

Скрипт **`scripts/rebuild-character-pages.ps1`** копирует текущие файлы из **`info/characters/`** в **`content/characters-archive/`** и генерирует новое тело из существующего frontmatter **на месте** в **`info/characters/*.md`**.
