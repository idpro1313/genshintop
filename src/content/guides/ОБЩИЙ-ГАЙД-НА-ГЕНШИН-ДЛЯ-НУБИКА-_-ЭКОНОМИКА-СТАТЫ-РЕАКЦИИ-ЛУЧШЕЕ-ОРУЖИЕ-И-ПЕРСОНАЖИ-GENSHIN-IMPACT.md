## Введение

Данный гайд представляет собой универсальное руководство для игроков Genshin Impact, которые хотят разобраться в основных механиках игры. Здесь вы найдете информацию об экономике, характеристиках персонажей, элементарных реакциях, лучшем оружии и рекомендации по сборке команд.

## Экономика игры

### Мора

**Мора**— основная игровая валюта, необходимая для:**
- Прокачки уровня персонажей
- Возвышения персонажей
- Прокидки талантов
- Улучшения артефактов
- Возвышения оружия

**Источники получения:**
- Сундуки на карте
- Награды за выполнение заданий
- Награды за повышение ранга приключений
- Подземелья (особенно Артерия Земли)
- Еженедельные боссы
- Обмен в магазине Паймон

**Советы по экономии:**
- Не тратьте мору на прокачку 4-звездочных артефактов до позднего этапа игры
- Сначала собирайте необходимые материалы, потом тратьте мору на улучшение
- Используйте консервированную смолу в конце недели для максимальной эффективности

### Примогемы

**Примогемы**— валюта для получения новых персонажей и оружия через систему молитв (гача).**

**Источники получения:**
- Сюжетные задания
- Ежедневные поручения (4 задания в день = ~60 примогемов)
- Подземелья
- Сундуки
- Награды за повышение ранга приключений
- Еженедельные боссы
- Бездна (Абисса)
- События
- Интерактивная карта (метки анемокулов, геокулов и др.)
- Промокоды от разработчиков

**Важно:** Примогемы — это ограниченный ресурс. Копите их на персонажей, которых хотите получить, а не тратьте на стандартный баннер.**### Смола**Смола**— основной ресурс для фарма в игре. Используется для:**
- Прохождения подземелий
- Сражения с боссами
- Сбора ресурсов с Артерии Земли

**Механика:**
- Максимальный запас: 200 единиц
- Восстановление: 1 единица за 8 минут
- Полное восстановление с нуля: ~26 часов 40 минут

**Консервированная смола (сгущёнка):**
- Можно получить за повышение ранга приключений
- Награды в магазинах регионов
- События
- Промокоды
- Даётся в день рождения

**Когда тратить смолу:**
- До 50 ранга приключений: тратьте только накопившуюся смолу, не консервированную
- После 50 ранга: открывайте консервы для фарма на высокой сложности подземелий
- Фармите Артерию Земли для книжек опыта и моры
- Боссов — только для необходимых материалов возвышения

## Характеристики персонажей

### Базовые характеристики

| Характеристика | Описание | На что влияет |
|----------------|----------|---------------|
| **Уровень** | Базовый уровень персонажа | Базовые статы, доступ к контенту |
| **HP (Здоровье)** | Количество жизней персонажа | Выживаемость, урон от некоторых навыков |
| **Сила атаки (АТК)** | Базовый урон персонажа | Урон от атак и навыков |
| **Защита (DEF)** | Сопротивление урону | Получаемый урон |
| **Выносливость**| Шкала для бега, плавания, заряженных атак | Мобильность и урон заряженных атак |**Как повышать базовые характеристики:**
- Уровень — через материалы возвышения и мору
- Выносливость — через подношения статуям Архонтов в Мондштате и Лиюэ

### Вторичные характеристики

| Характеристика | Описание | Рекомендуемые значения |
|----------------|----------|------------------------|
| **Крит-шанс** | Шанс нанести критический урон | 60-70% (для большинства ДД) |
| **Крит-урон** | Множитель критического урона | 120-150%+ |
| **Бонус элементального урона** | Усиление урона определённой стихии | 46-80%+ (зависит от персонажа) |
| **Мастерство стихий (МС)** | Усиление урона элементарных реакций | Зависит от реакции и сборки |
| **Восстановление энергии** | Скорость зарядки взрыва стихии | 100-130%+ (зависит от персонажа) |
| **Бонус лечения** | Усиление исцеления | Для целителей |
| **Прочность щита**| Эффективность щитов | Для щитовиков |**Формула урона:**

```
Итоговый урон = (Базовый урон × (1 + Бонус АТК)) × (1 + Бонус элементального урона) × (1 + Крит-урон, если крит)
```

**Золотая середина для критов:**
- 70% крит-шанса на 140% крит-урона
- Соотношение 1:2 (шанс:урон)

## Сборка персонажей

### Таланты

**Определение основной характеристики:**
- Если в талантах указано HP — собирайте в HP
- Если указана защита — собирайте в защиту
- Если указана сила атаки — собирайте в АТК
- Решение принимается на основе описания талантов

**Приоритет прокачки:**
- Мейн ДД: Обычные атаки > Взрыв стихии > Элементальный навык
- Саб ДД/Саппорт: Взрыв стихии > Элементальный навык > Обычные атаки

**От чего зависит выбор:**
- Роль персонажа в отряде
- Какие кнопки отыгрывают основную роль

### Артефакты

**Типы артефактов:**
- Цветок — даёт только HP
- Перо — даёт только силу атаки
- Часы — различные статы (АТК%, HP%, DEF%, Восстановление энергии, МС)
- КубоК — различные статы + бонус элементального/физического урона
- Шапка — различные статы + крит-шанс/крит-урон/бонус лечения

**Сеты артефактов:**
- 2+4 (два предмета из одного сета, четыре из другого) — наиболее гибкий вариант
- 4+0 (полный сет) — если бонус сета очень сильный

**Основные статы для разных ролей:**

| Роль | Основные статы | Субстаты |
|------|----------------|----------|
| ДД от АТК | АТК%, Криты, Бонус элемента | АТК, Восстановление энергии |
| ДД от HP | HP%, Криты, Бонус элемента | HP, Восстановление энергии |
| ДД от DEF | DEF%, Криты | DEF, Восстановление энергии |
| Саппорт | Восстановление энергии, HP%/DEF% | зависит от функции |
| Целитель | Бонус лечения, HP% | HP |

## Элементарные реакции

В игре есть 7 элементов: Анемо, Гео, Электро, Дендро, Гидро, Пиро, Крио.

### Реакции первого порядка

Реакции, которые наносят урон и могут критовать.

| Реакция | Элементы | Описание |
|---------|----------|----------|
| **Заморозка** | Гидро + Крио | Не наносит урона, но обездвиживает врага |
| **Рассеивание** | Анемо + Пиро/Крио/Электро/Гидро | Наносит урон + понижает сопротивление |
| **Заряжен** | Электро + Гидро | Наносит урон |
| **Перегруз** | Электро + Пиро | Наносит урон + небольшой отброс |
| **Сверхпроводник** | Электро + Крио | Наносит урон + понижает физ. защиту |
| **Горение** | Пиро + Дендро | Наносит урон |
| **Бутонизация** | Дендро + Гидро | Создаёт бутон, который взрывается |

### Реакции второго порядка

Реакции, которые НЕ могут критовать и зависят от мастерства стихий.

| Реакция | Элементы | Описание |
|---------|----------|----------|
| **Кристаллизация** | Гео + Пиро/Крио/Электро/Гидро | Создаёт щит |
| **Лунный кристалл** | (с Линнеей) | Создаёт тотемы |
| **Разбивание**| (дополнительный урон к реакциям) | Активируется на бутон |**Важные формулы:**

*Реакции первого порядка:*
```
Урон реакции = (Базовый урон реакции × Уровень) × (1 + МС / (МС + 1200))
```

*Реакции второго порядка:*
```
Урон реакции = Базовый урон × Уровень персонажа × Бонус от МС
```

**Какие реакции лучше:** 1.**Бутонизация**— максимальный урон на минимальных вложениях**2.**Заморозка**— контроль + высокий крит-шанс через набор «Заблудший в метели»**3.**Рассеивание**— универсальный урон + срез сопротивлений**Что НЕ рекомендуется:**
- Реакции первого порядка с Гео (Кристаллизация) — не наносит урон
- Горение в чистом виде — сложно использовать эффективно

## Рекомендации по персонажам

### Лучшие Мейн ДД (на момент записи)

1. **Хутао** — Пиро ДД, играется от HP, высокий урон
2. **Синьо (Синрю)** — Гидро ДД, играется от АТК
3. **Арлекина** — Пиро ДД, играется от HP
4. **Дэхья** — Пиро ДД, играется от HP
5. **Нивелла (Ниветта)** — Крио ДД, играется от АТК

### Лучшие саб ДД

1. **Синьо (саб)** — универсальный саб
2. **Беннет** — лучший буфер атаки
3. **Сахароза** — лучший бафф МС
4. **Казуха** — бафф МС + рассеивание

### Лучшие саппорты/щитовики

1. **Чжун Ли** — лучший щит + контроль
2. **Беннет** — лечение + бафф
3. **Кокоми** — лечение + Гидро
4. **Диона** — щит + лечение + Крио

### Лучшие бесплатные персонажи

1. **Делюк** — отличный ДД (даётся бесплатно)
2. **Синьо** — можно получить через события
3. **Беннет** — один из лучших саппортов
4. **Лизу, Кайю, Эмбер** — базовые персонажи
