# Ивенты Genshin Impact 6.x: текущие события и расписание

Ивенты Genshin Impact — ограниченные по времени активности с уникальными наградами: примогемами, бесплатными 4★ персонажами, материалами возвышения и эксклюзивными декорациями для Чайника. Этот гайд — общий хаб по событиям актуального патча 6.x.

## Где смотреть актуальное расписание

Точные даты и список ивентов всегда есть в:

- официальной [новостной ленте HoYoverse](https://genshin.hoyoverse.com/news);
- [HoYoLAB](https://www.hoyolab.com/) — пост «Event Compendium» в начале каждого патча;
- внутри игры: вкладка «События» в меню paimon → Manuals.

## Какие типы ивентов бывают

- **Главные сюжетные** — «зонтик» патча, привязан к региональной арке. Дают 420–500 примогемов, костюм / 4★ персонажа.
- **Боевые** — Театр Воображариума, временные подземелья, BOSS-rush. Награды: примогемы, мора, материалы.
- **Исследовательские** — путешествия по новой локации, сбор Окул и пятиглазиков.
- **Мини-игры** — рыбалка, гонки, кулинария, фотомод. Низкая сложность, расслабленный темп.
- **Логин-кампании** — простые ежедневные награды без активной игры.

## Награды и приоритеты

1. Сначала забрать **бесплатного 4★ персонажа** или **постоянное оружие**, если оно есть в патче.
2. Дальше — **примогемы**: каждый ивент даёт 420–800, что эквивалентно 2–5 молитвам.
3. **Материалы возвышения и Книги опыта** — приятный бонус, если идут к нужному персонажу.
4. Декор для Чайника и репутация — по желанию.
