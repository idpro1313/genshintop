## Полезности на Старте

### Первый сундук

Сразу после начала:
- Поверните НАЛЕВО на берегу
- Доплывите до камней
- Заберите сундук с примогемами

### Роскошный сундук в Мондштате

1. Поднимитесь на главную площадь
2. Поднимитесь по лестнице СПРАВА
3. Прыгайте по бордюрам до конца
4. Получите роскошный сундук!

### Сундук у сестры Виктории

1. Зайдите в собор
2. Поговорите с сестрой Викторией
3. Выбирайте ответы, где появляется сундучок
4. Получите 4-звёздочный артефакт

## Выносливость

**Выносливость** нужна для:
- Бега (дольше)
- Плавания
- Заряженных атак
- Лазания

**Как увеличить:**
- Собирайте анемокулы в Мондштате
- Собирайте геокулы в Лиюэ
- Собирайте электрокулы в Инадзуме
- И так далее для всех регионов

## Элементы и Реакции

### 7 элементов

| Элемент | Цвет | Особенности |
|---------|------|-------------|
| Пиро | Красный | Высокий урон |
| Гидро | Синий | Лечение, контроль |
| Крио | Голубой | Заморозка, криты |
| Электро | Фиолетовый | Энергия, реакции |
| Анемо | Зелёный | Рассеивание |
| Гео | Жёлтый | Щиты |
| Дендро | Зелёный | Бутоны |

### Основные реакции

- **Заряжен** (Электро + Гидро) — урон
- **Перегруз** (Электро + Пиро) — урон + отброс
- **Сверхпроводник** (Электро + Крио) — урон + срез физ. защиты
- **Рассеивание** (Анемо + любой) — урон + снижение сопротивления
- **Заморозка** (Гидро + Крио) — контроль

## Энергия

**Для чего нужна:**
- Для использования взрыва стихии (Q)
- Это основная способность персонажей

**Как получать:**
- Убийство врагов
- Использование навыка (E)
- Урон по врагам
- Сбор частиц

**Совет:** Собирайте восстановление энергии для саппортов!

---

## Характеристики

### Основные

- **HP** — здоровье
- **АТК** — сила атаки
- **Защита** — сопротивление урону
- **Выносливость** — для бега, плавания

### Вторичные

- **Крит-шанс** — шанс крита (рекомендуется 60-70%)
- **Крит-урон** — множитель крита (рекомендуется 120-150%)
- **Бонус элементального урона** — усиление стихии
- **Мастерство стихий** — усиление реакций
- **Восстановление энергии** — скорость зарядки

---

## Артефакты

### Типы

- **Цветок** — только HP
- **Перо** — только АТК
- **Часы** — можно выбрать стат
- **Кубок** — стат или бонус элемента
- **Шапка** — стат или криты/хил

### Сеты

- 4 предмета = полный бонус
- 2+2 = комбинация двух сетов
- Выбирайте под элемент персонажа

### Когда фармить

**До AR 45** — просто используйте любые 5★
**После AR 45** — фармите конкретные сеты

## Отряды

### Роли

- **Мейн ДД** — основной урон на поле
- **Саб ДД** — урон «из кармана»
- **Саппорт** — помощь отряду
- **Целитель/Щитовик** — выживаемость

### Формула

```
Мейн ДД + Саб ДД + Саппорт + Целитель
```

## Заключение

Запомните главное:

1. **Сундуки на старте** — бесплатные примогемы
2. **Выносливость** — собирайте кулы
3. **Реакции** — используйте их правильно
4. **Энергия** — важна для ультимейтов
5. **Артефакты** — фармите после AR 45
6. **Сбалансированный отряд** — залог успеха
