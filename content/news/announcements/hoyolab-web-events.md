HoYoLAB Web Events — мини-ивенты в браузере на [hoyolab.com](https://www.hoyolab.com). Запускаются параллельно с большими игровыми ивентами; не требуют запуска игры.

## Типичные форматы

| Формат | Награда | Длительность |
|--------|---------|--------------|
| **Daily Check-in** (24-day) | ~60-180 примо + 4★ материалы | Каждый патч |
| **Memory Quiz** | ~40-60 примо | 1-2 недели |
| **Pre-Patch Speculation** (Will-it-Banner) | 40 примо | До патча |
| **Anniversary Web Event** | ~600-1000 примо + Wings | 1-2 недели в Сентябре |
| **Article-writing** (длинные эссе) | Acquaint Fates, Mora | По решению HoYoverse |

## Сколько даёт за патч

В среднем **300-500 примогемов** в месяц через все web events. Это эквивалент 2-3 wishes, но требует только 5-10 минут на регулярный check-in.

## Регистрация

1. Зайти на [hoyolab.com](https://www.hoyolab.com).
2. Войти через тот же HoYoverse аккаунт, что и в игре.
3. Привязать UID персонажа (чтобы получить награды в внутриигровой Mail).
4. Перейти в **Events** → выбрать актуальный.

## Daily Check-in особенно важен

Месячный check-in (24 дня подряд) даёт суммарно **~300 примогемов + 5 Hero's Wit + 7 Mystic Enhancement Ore + 50000 Mora**. Пропуск дня _не_ обнуляет счёт — можно «catch-up» в течение месяца.
