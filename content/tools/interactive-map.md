Интерактивная карта — главный «гид по местности» Тейвата. Без неё сложно собирать диковины, открывать сундуки и проходить мировые квесты.

## Что должна уметь карта

- Слои: сундуки (по рангам), оракулы, статуи, диковины (с фильтром по типу), боссы, домены, респаун-точки.
- Поиск по тексту: «куда положили Lumitoile №14».
- Маркеры пользователя: чтобы отметить ещё неразгаданные паззлы.
- Привязка к аккаунту: «уже собрано — скрыть».

## Сторонние эталоны

- **genshin.gg/map** — самая полная по слоям, бесплатная.
- **mapgenie.io/genshin-impact** — альтернативная подача, требует регистрации для прогресса.
- **Hoyolab Interactive Map** — официальная, удобна на мобильном.

Пока внутреннего инструмента нет — рекомендуем genshin.gg/map для PC и Hoyolab для телефона.
