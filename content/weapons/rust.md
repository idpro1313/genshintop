Rust — лучший 4★ лук для DPS, чей урон в обычных автоатаках (без прицельных).

## Характеристики

| Параметр | Значение |
|----------|----------|
| Базовая атака (90) | 510 |
| Доп. характеристика | ATK% 41.3% |

## Пассив

«Rapid Firing»: бонус урона обычных стрел +40% (R1, до +80% R5). Бонус урона прицельных стрел −10%.

## Кому подходит

- **Ёимия** — best 4★ для пар-Ёимии.
- **Файрру́зан** — топ для саппорта.
- **Чевреза** — рабочий.
- **Эмбер** — приличный (но Эмбер обычно не основной DPS).

## Альтернативы

- **The Stringless** — для саппорта на EM.
- **Sacrificial Bow** — для энерго-саппорта.
- **Polar Star** (5★) — для Тарта́рьи.
