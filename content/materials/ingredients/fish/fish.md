Fish — общая категория рыбы как **ингредиента готовки**. Получают двумя способами:
1. **Fishing System** (с 2.1): удочка + bait в специальных fishing point. См. [`/guides/basics/fishing`](/guides/basics/fishing).
2. **Кулинарная рыба** (продажа NPC + дроп редких морских животных).

## Конкретные виды (для готовки)

| Вид | Регион | Bait |
|-----|--------|------|
| **Medaka** | Mondstadt | Fruit Paste |
| **Crystalfish** | Mondstadt пещеры | Fruit Paste |
| **Lunged Stickleback** | Liyue лагуны | Redrot Bait |
| **Pufferfish** | Liyue побережье | Fruit Paste |
| **Sweet Flounder** | Inazuma побережье | Fake Fly |
| **Salmon** | Mondstadt-Liyue реки | Fake Fly |
| **Raimei Angelfish** | Inazuma (Watatsumi) | False Worm |

## Применение в готовке

| Блюдо | Тип |
|-------|-----|
| **Tea-Break Pancake** | HP recovery |
| **Vegetarian Abalone** | DEF Boost |
| **Lotus Seed and Bird Egg Soup** | HP recovery |
| **Black-Back Perch Stew** | HP recovery |
| **Sticky Honey Roast** | HP recovery + ATK Boost (Сахароза specialty) |
| **All-Weather Beauty** (Кадзуха specialty) | -25% sprint stamina |
| **Adeptus' Temptation** | ATK Boost (mighty) |

## Также ингредиенты

- **Fish Paste**: crafted из 1 Fish + 1 Salt → используется для bait crafting.
- **Sashimi Platter**: рассольное блюдо для DEF.
