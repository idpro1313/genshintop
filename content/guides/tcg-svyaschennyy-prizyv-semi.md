# Священный призыв семи: гайд по TCG в Genshin Impact

Священный призыв семи (Genius Invokation TCG) — карточная игра внутри Genshin Impact. Мини-режим с собственным игровым полем, колодой, очками чести и серией NPC-противников. Этот гайд — стартовая точка: правила, типы карт, основные механики и тиры колод.

## Базовые правила

- **Колода:** 3 карты персонажей + 30 карт действий.
- **Цель:** уничтожить все 3 карты персонажа противника (HP=0).
- **Раунд:** игроки бросают **кубики стихий** (8 шт.), используют их для оплаты способностей и карт действий, по очереди ходят и переключают активного персонажа.
- **Усиления:** комбо-атаки, элементальные реакции (как в основной игре), уникальные пассивки персонажей.

## Типы карт

- **Карты персонажей** — основа колоды. У каждого 3 способности (обычная атака, элементальный навык, взрыв) и собственное HP.
- **Снаряжение / артефакты** — постоянные баффы (увеличение урона, лечение, доп. кубики).
- **Поддержка** — оперируют в зоне поддержки, дают эффекты на несколько раундов.
- **События / Еда** — одноразовое действие (доп. кубик, исцеление, кросс-эффект).

## Что качать сначала

1. Освойте **базовый набор карт** через NPC «Прехдок» в Мондштадте.
2. Возьмите **бесплатных персонажей** из туториалов (Фишль, Янь Фэй, Сян Лин, Барбара, Беннет).
3. Получите **колоды у NPC-противников** в Ли Юэ и Инадзуме — они дают бесплатные стартовые мета-карты.
4. Перейдите к **серии турниров «Чемпионская защита»** для очков чести и эксклюзивных карт.

## Тир-лист популярных колод (актуальный)

- **S — стабильно сильные:** Сян Лин + Беннет + Кадзуха (Воспламенение), Танчжуй + Райдэн + Хайн (Сверхпроводник), Нахида + Сян Лин + Беннет (Дендро-Воспламенение).
- **A — сильные с условиями:** Гань Юй + Сян Лин (Расплавление), Аратаки Итто + Альбедо (Гео-моно), Айрин (Айра) + Шеврёз + Кадзуха.
- **B — играбельны в PvE-турах:** Кэйа + Барбара + Дилюк, Янь Фэй + Шиная + Беннет, Тигнари + Фишль + Кокоми.

## Награды и зачем играть

- 1500+ примогемов с прохождения всех туториалов и серий NPC.
- Эксклюзивные предметы для Чайника и фотомод.
- Карты-достижения, открывающие подсказки по сюжету и лору.
