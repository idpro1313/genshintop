# Тир-лист персонажей Genshin Impact 6.x

Тир-лист обновлён под актуальный патч 6.x. Все 5★ персонажи рассматриваются на C0 с топовым 4★ оружием, 4★ — на C6 с подходящим 4★ оружием. Учтены **Витая Бездна** и **Театр Воображариум** конца Натлана / начала Снежной.

Если ищете срез меты конца Фонтейна — оставлен [архивный тир-лист 4.8](/guides/tier-list-4-8).

## Tier S — определяют мету

- **Главный DPS:** Мавуика, Невилетт, Райдэн Сёгун, Сяо, Ху Тао, Аль-Хайтам
- **Саб-DPS:** Сян Лин, Фишль, Юй Цзинь, Шеврёз, Куки Синобу
- **Саппорт / баффер:** Беннет, Кадзуха, Нахида, Фурина

## Tier A — топ при правильной команде

- **Главный DPS:** Хутао, Айола, Кэ Цин, Ёимия, Аратаки Итто, Кинич
- **Саб-DPS:** Йелан, Сян Лин (без C6), Сяо без саппорта Фурины
- **Саппорт:** Чжун Ли, Кокоми, Бусил, Дилюк-копья (Гейо), Барбара (Гидро-аппликация)

## Tier B — играбельны в нишевых командах

Работают, но требуют либо созвездий, либо специфической команды. Барбара, Розария, Хайзенберг, Рэйзор, Сахароза, Мона, Альбедо, Лиза, Тома, Чарка.

## Tier C — нечасто оптимальны

Эмбер, Кэйа, Сян Сэнь, Гайя, Лиза без C6. Берите из эмоциональной привязанности — игра позволяет.

## Команды-сборки актуальной меты

- **Натлан-меле:** Мавуика + Цитлали + Шеврёз + Беннет
- **Сверхрасцветание:** Райдэн / Сайно + Нахида + Фурина + Кадзуха
- **Бутонизация:** Нилу + Нахида + Йелан + Куки Синобу
- **Воспламенение:** Аль-Хайтам + Беннет + Сян Лин + Кадзуха
- **Заморозка:** Айола + Кокоми + Шеврёз + Кадзуха
- **Гипербл уум:** Райдэн + Кадзуха + Беннет + Сян Лин (классика без Дендро)
