Local Specialties (региональные ингредиенты) — растения, минералы и природные объекты, специфичные для конкретного региона Тейвата. Каждому играбельному герою нужно **168 единиц** Local Specialty его родного региона для полной ascension (1→90).

## Ascension cost

Точная разбивка для одного героя:

| Ascension фаза | Local Specialty |
|----------------|-----------------|
| 20→40 | 3 |
| 40→50 | 10 |
| 50→60 | 20 |
| 60→70 | 30 |
| 70→80 | 45 |
| 80→90 | 60 |
| **Итого** | **168** |

(плюс ascension-gem, boss-материалы, Mora.)

## Когда фармить

- **Респаун: 48 часов** real-time (после сбора).
- **Серверный сброс не влияет** — таймер от момента сбора.
- **Лучшее время:** перед каждой ascension фазой персонажа (за 2-3 дня собрать всё нужное).

## Маршруты фарма

Самое эффективное — построить **route через все локации** региональной specialties за один заход:

1. Открой [Interactive Map](/tools/interactive-map) → фильтр specialties.
2. Поставь маркер на Cecilia (для Кадзухи) → **41 локация в Мондштадте**.
3. Раздели на 2-3 захода (по 14 локаций × 3 дня = 42 за неделю).
4. Используй стамина-героев (Янь Фэй, Сахароза, Кадзуха с anti-cold buffs).

## Tips

1. **Кадзуха > Сахароза** для group-pick — Anemo-вращает много specialties в один Burst.
2. **Янь Фэй F2P** — стамина-pick с Tornado.
3. **Cooking buffs:** Northern Smoked Chicken (-15% stamina) — must для long-routes.
4. **Anti-cold (Драгонспайн):** Goulash перед stamina-фармом в холоде.
