Сундуки — основные источники Primogems, Mora, Hero's Wit и оружия в открытом мире. **Открытие сундука даёт примогемы один раз навсегда** (он не респавнится).

## 5 типов сундуков

| Тип | Цвет | Primogems | Mora | Прочие награды |
|-----|------|-----------|------|----------------|
| **Common Chest** | Светло-коричневый | 0-2 | 200 | Random |
| **Exquisite Chest** | Тёмно-коричневый | 2 | 1000 | Hero's Wit, Mystic Ore |
| **Precious Chest** | Голубой/фиолетовый | 5 | 2500 | 3★/4★ артефакт, Hero's Wit |
| **Luxurious Chest** | Золотой | 10 | 5000 | 4★ артефакт, материалы |
| **Remarkable Chest** (с 2.0) | Зелёный (Inazuma+) | Зависит | Зависит | За Puzzle / Combat (специальный) |

## Где найти

- **Open world:** ~3000+ сундуков по всем регионам.
- **Domain Trial:** перед боем в Domain — сундук с артефактами (Common).
- **Boss arenas:** Hilichurl Camp boss-mob → Common/Exquisite.
- **Puzzles:** Geo Anomaly, Anemo Stone Tablet, Electro Cube, Dendro Memory.
- **Investigation:** иконка «?» (loose stones, скрытые ниши).
- **Quests:** многие квесты дают сундуки в финале.

## Респаун (что обновляется)

| Что | Респаун |
|-----|---------|
| **Сундуки** | НИКОГДА (одноразовые) |
| **Враги (мобы)** | 12-72 часа (зависит от типа) |
| **World Bosses** | Каждые ~24 часа после Resin-fight |
| **Local Specialties** (Cecilia, Glaze Lily) | **48 часов** |
| **Mining ore** (Crystal Chunk, Magical Crystal) | **48-72 часа** |
| **Cooking ingredients** (Apples, Mushrooms) | **24 часа** |
| **Furniture materials** (wood, paint) | **48-72 часа** |
| **Crystal Marrow / Onikabuto** | **48 часов** |

**Важно:** для local specialties (под персонажа) нужно фармить **2-3 локации** + **48-часовой ротейшн**. Карта помогает спланировать. См. [`/tools/interactive-map`](/tools/interactive-map).

## Приоритет открытия (новый регион)

При входе в новый регион:
1. **Открыть Statues of Seven** — телепорт + анемокулы/гео́кулы и т.д.
2. **Активировать teleporters** — телепортация сети.
3. **Сбор oculi (анемокулы и т.п.)** — для апгрейда статуй (HP, Stamina, Adventure EXP).
4. **Quests** — основные ивенты/архонт.
5. **Точечный фарм сундуков** — через Interactive Map.

## Tools для сундуков

- **HoYoLAB Interactive Map** ([webstatic.hoyoverse.com](https://webstatic-sea.hoyoverse.com/app/ys-map-sea/index.html)) — все сундуки с фильтрами по статусу «открыт/нет».
- **Genshin Center** ([genshin-center.com](https://genshin-center.com)) — счётчик сундуков по региону.
