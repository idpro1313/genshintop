Элементальная реакция возникает, когда вторая стихия попадает на цель, у которой уже висит «аура» от первой. Реакции делятся на **усиливающие** (пар, расплавление, аггравация, стимул, сжигание) и **трансформирующие** (заморозка, перегрузка, сверхзаряд, бурление, цветение, заряд, swirl, кристалл).

Усиливающие умножают урон таланта-триггера. Трансформирующие наносят отдельный «реакционный» урон, который зависит только от уровня героя и Мастерства Стихий.

## Основные реакции

### Пар (Vaporize) — Пиро + Гидро

Цель с гидро + пиро = ×2.0 к урону пиро-триггера. Цель с пиро + гидро = ×1.5 к урону гидро-триггера. Усиливающая.

Кто использует: Ху Тао + Е Лань / Сянлин + Беннетт + Сянлин + Бэннетт (стандарт). Гидро-носители: Е Лань, Кокоми, Сян Лин-нет, гидро-Сянлин-нет — Гидро-Сянлин не существует, исправлено: гидро-партнёром берут Е Лань, Сянгуй (нет), Кокоми, Чжон Юнь (нет — крио), Тарталья.

### Расплавление (Melt) — Пиро + Крио

Крио + пиро (триггер пиро) = ×2.0 к урону пиро. Пиро + крио (триггер крио) = ×1.5 к урону крио. Усиливающая.

Кто использует: Гань Юй + Беннетт + Сянлин + Кадзуха («Расплавление-Гань Юй»), Айяка-Расплавление (редко), Дилюк-Расплавление с Шэнь Хэ.

### Заморозка (Freeze) — Гидро + Крио

Цель замораживается на 1–4 секунды (зависит от элементальной массы аур). Не наносит урона сама, но обездвиживает цель. Идеально работает с сетом «Снежная странница» (+15% крит-шанса по мокрым/замёрзшим).

Кто использует: Айяка + Кокоми + Сяонлин (нет — крио). Стандарт: Айяка + Шэнь Хэ + Кокоми + Кадзуха.

### Перегрузка (Overload) — Пиро + Электро

Взрыв пиро-урона. Не масштабируется по уровню героя. Ключевой эффект — отбрасывание лёгких врагов и ноки.

Кто использует: редко, в специфических билдах Кли + Бэй Доу.

### Сверхзаряд (Superconduct) — Крио + Электро

Импульс крио-урона + −40% физического резистенса противника на 12 сек. Открывает физ-билды (Эола, Эмбер).

Кто использует: Эола + Бэннетт (электро + крио — нет; Эола + Шэнь Хэ + Райдэн).

### Заряд (Electro-Charged) — Гидро + Электро

Висит цикл электро-уронов на промокших целях. Передаётся по цепочке. Хорошо для AoE-ситуаций.

Кто использует: Бэй Доу + Е Лань + Сянлин (Заряженный Бэй Доу — старый бенч-mark).

### Аггравация (Aggravate) — Дендро + Электро

Цель с дендро-аурой + электро-триггер = добавочный электро-урон, масштабируется по уровню героя + бонус электро-урона. Усиливающая.

Кто использует: Кэ Цин-Дендро, Алхайтам, Скирк, Райдэн-Аггравация (опционально).

### Стимул (Quicken) → Бурление (Hyperbloom) и Сжигание (Burgeon)

Дендро + Электро на цели создаёт «квикен» (видимая аура). С него:

- **Бурление (Hyperbloom):** ядро бурления взрывается с электро-уроном. Триггер — электро (часто Куки Синобу или Райдэн).
- **Цветение (Bloom):** Дендро + Гидро = ядро цветения, которое взрывается с дендро-уроном через 6 сек или при триггере.
- **Сжигание (Burgeon):** ядро цветения + пиро = AoE-пиро-взрыв. Триггер — обычно Тома, Беннетт.

### Swirl — Анемо + любая основная стихия

Передаёт ауру первичной стихии на соседних врагов с уроном анемо. С сетом «Изумрудная тень» дополнительно сбивает резистенс цели к этой стихии.

Кто использует: Кадзуха, Сахароза, Хейцзо, Странник, Венти.

### Кристалл (Crystallize) — Гео + основная стихия

Создаёт осколок щита нужной стихии. Щит даёт +25% соответствующего стихийного резистенса. Часто работает в гео-командах с Альбедо или Чжон Ли.

## Сжигание (Burning)

Дендро + Пиро = постоянный пиро-урон, пока висит дендро-аура. Часто избегаемая реакция в дендро-билдах: пиро «съедает» дендро-ауру и ломает цветение.

## Что взять из этого

1. Под одну команду — 1–2 типа реакций. Нет смысла «всех стихий по одной».
2. Усиливающие реакции (пар, расплавление, аггравация) — масштабируются по статам триггера: ставьте крит и бонус стихии.
3. Трансформирующие реакции (заморозка, перегрузка, бурление, цветение) — масштабируются по Мастерству Стихий и уровню. Сборка артефактов другая.
