## Гарантированный 5★ rate-up (worst-case)

- **Худший случай:** провал 50/50 на 90 → ещё 90 на гарантированного = **180 wishes = 28800 primogems**.
- **Средний случай:** 50% × 62 + 50% × (62 + 62) = **93 wishes = 14880 primogems**.

## Weapon Banner (Epitomized Path)

| Pity (Weapon) | Soft-pity start | Hard-pity |
|----------------|-----------------|-----------|
| Базовая 5★ | 0.7% | — |
| Soft-pity начало | **63** | — |
| Hard-pity | **80** | 100% |
| Expected wishes до 5★ | **~52** | — |

**Epitomized Path:** выбираешь 1 из 2 rate-up 5★. После 2 «провалов» (получения не выбранного 5★) — гарант на выбранное.

| Сценарий | Wishes до выбранного 5★ |
|----------|---------------------------|
| Лак (пятёрка с 1-й попытки) | 52 |
| Среднее (1 fate-point) | 104 |
| Худшее (2 fate-points → гарант) | **156** |
| **Среднее с учётом всего** | **~120 wishes = 19200 primogems** |

Поэтому Weapon Banner **дороже Character Banner** в среднем на ~30%.

## Plan-3-patches strategy (F2P-friendly)

```
Patch X.0: пропустить (копить ~5500 примо).
Patch X.1: пропустить (копить ~11000 примо).
Patch X.2: крутить (стартовый pool 11000 = 68 wishes).
   - С учётом soft-pity ~62 → почти гарантированный 5★ rate-up.
```

## Chronicled Banner (с 4.5)

Раз в 2-3 патча Chronicled Banner:
- Выбираешь 1 из ~30 старых 5★ (Чжон Ли, Венти, Гань Юй, Кадзуха, Райдэн, Аяка, Найхида, Невиллет, Фурина и т.п.).
- Pity такой же (90, soft с 74).
- 50/50: 50% rate-up выбранный, 50% — другой 5★ из pool.
- **Не существует «гарант» на выбранного** — может 50/50-провал даст всё равно ценный 5★ из pool, но не выбранного.
- **Когда крутить:** когда нужен старый 5★, недоступный на event-баннерах.

**Tip:** провал 50/50 на Chronicled выпадает в _случайного_ 5★ из pool — это всё равно полезно (старые мета-герои).

## Optimal F2P-стратегия

1. **Welkin Moon** ($5/mo) — лучший single value-pick.
2. **Стандартный баннер** (Wanderlust) — крутить **только Acquaint Fates**, не Intertwined.
3. **Beginner банner** — обязательно (Noelle гарант).
4. **Lantern Rite/Irodori/Sabzeruz** — забрать free 4★.
5. **Spiral Abyss / Imaginarium Theatre** — main source примо вне коммисcий.
6. **Daily check-in HoYoLAB** — +60 к балансу.
7. **Чтение Special Program codes** — 300 примо/patch.
