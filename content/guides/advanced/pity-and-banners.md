## Базовые числа

- Персонаж 5★: жёсткий питти на 90 крутке. Мягкий — с 75-й, шанс растёт.
- Оружие 5★: жёсткий 80, мягкий с 65.
- 4★: 10-крутка гарантирует 4★.

## Гарант 50/50

- На событийном баннере персонажа: первый 5★ может быть «потерянным» (стандартный 5★). Если так, следующий 5★ — гарантированно событийный.
- На стандарте 50/50 нет — выпадает любой стандартный 5★.

## Epitomized Path (оружие)

- В баннере оружия выбираете один из двух 5★. Каждая «потеря» 5★ (выпал не выбранный) даёт 1 «фатум»: после 2 фатумов следующий 5★ обязательно ваш выбранный.
- Между патчами фатумы сохраняются ТОЛЬКО внутри одного баннера. Сменился баннер — фатумы обнулены.

## Хроника (Chronicled Wish)

Раз в полупатч появляется баннер «Хроника». Можно выбрать один из десятков «старых» 5★ персонажей (без оружия). Питти с обычного баннера не переносится.

См. также [Каталог баннеров](/news/banners).
