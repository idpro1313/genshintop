Фишль — самозванная «Принцесса Осуждения» из Иммернахтрейха в сопровождении вороньего фамильяра Оз. В игре — лучший off-field электро-сабдамаг 4★ для Hyperbloom, Aggravate и Quicken-команд.

## Кратко

- **Стихия:** Электро.
- **Оружие:** Лук.
- **Регион:** Мондштадт.
- **Релиз:** 1.0.

## Роль

- **Сабдамаг** через Оз: каждые 1 сек электро-удар по ATK.
- **Аппликатор электро** для Aggravate, Hyperbloom, Overload.
- **C6** даёт прокачку до полноценного DPS-альтернативы.

## Таланты

| Талант | Назначение |
|--------|------------|
| NA: Bolts of Downfall | Стандартный лук |
| E: Nightrider | Призывает Оз: бьёт каждые 1 сек до 10 сек |
| Q: Midnight Phantasmagoria | Перемещение, AoE-урон, повторно вызывает Оз |
| A4 | NA с Оз даёт extra-удар |

## Созвездия

| C | Эффект |
|:-:|--------|

C6 — заметный апгрейд для Aggravate-DPS.

- **Сет:** Thundering Fury (4) или Golden Troupe (4).
- **Часы:** ATK%, **кубок:** Electro DMG%, **шапка:** Crit DMG / Crit Rate.
- **ER:** 130–150%.

## Команды

- Aggravate: Фишль + Райдэн + Накида + Беннетт (нет, Aggravate без пиро) → Фишль + Кено + Кадзуха + Накида.
- Hyperbloom: Фишль + Накида + Сюй Цю + Кокоми.
- Overload: Фишль + Сян Лин + Беннетт + Чжон Ли.
