Беннетт — стандартный пиро-меч 4★, с трагической судьбой невезучего исследователя. В мете — один из топ-3 саппортов всех времён: его Q даёт лечение и +1000 атаки в круге.

## Кратко

- **Стихия:** Пиро.
- **Оружие:** Меч.
- **Регион:** Мондштадт.
- **Релиз:** 1.0.

## Роль

- **Главная:** бафф атаки команды (+~1000 ATK в Q-круге).
- **Дополнительная:** лекарь (восстанавливает на основе HP%).
- **Сабдамаг:** через Е может бить за 100 пиро-сфер для энергии Сян Лина.

## Таланты

| Талант | Назначение |
|--------|------------|
| NA: Strike of Fortune | Стандартная мечовая комба |
| E: Passion Overload | Пиро-удар, при заряде 1/2 — комбо ударов |
| Q: Fantastic Voyage | Создаёт круг лечения и баффа атаки. Снимок при входе в круг |
| A1, A4 | Бонусы лечения и баффа |

## Созвездия

| C | Эффект |
|:-:|--------|
| C1 | E кулдаун −20% |
| C2 | Восстановление лечения |
| C3 | E +3 уровня |
| C4 | E лечит вне круга |
| C5 | Q +3 уровня |
| C6 | Бафф атаки игнорирует HP-кэп; добавляет пиро-инфузию мечам |

C6 — двухсторонний меч: на пиро-DPS отлично, на пар-команды Ху Тао иногда вредно (ломает ауру).

## Билд

- **Сет:** Noblesse Oblige (4★) — для бафа атаки команде.
- **Часы:** ER% или ATK% (зависит от ER без Q-баттереи).
- **Кубок:** ATK% или HP% (для бафа).
- **Шапка:** Лечение или Крит-урон (если C6 + autoatake).
- **ER требование:** 200%+ без Q-сатпорта.

## Оружие

| Оружие | Заметки |
|--------|---------|
| Aquila Favonia (5★) | Бафф ATK +20% и лечение |
| Festering Desire (4★, ивент) | ER + бонус Е-урона |
| The Flute (4★) | Хороший доп. урон |
| Favonius Sword (4★) | Для энергии команды |
| Mistsplitter (5★) | Если C6 — пиро-DPS |

## Команды

- Любая пиро-команда: Беннетт + Сян Лин + Сюй Гуй (нет, Xingqiu) + Кадзуха.
- Пар: Беннетт + Ху Тао + Е Лань + Чжон Ли.
- Расплавление: Беннетт + Гань Юй + Сян Лин + Кадзуха.

## Кому подойдёт

- Всем — это base-line саппорт.
- Кодек C0 хватает для 90% контента; C6 — только если у вас пиро-меч-DPS.
