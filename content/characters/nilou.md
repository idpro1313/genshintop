Нилу — главная танцовщица театра «Зубаджи» в Сумеру и кеншин знаменитой школы танца. В игре — гидро-меч 5★ с уникальным enabler-механизмом: при моно-гидро/дендро-команде её Q превращает Bloom-семена в супер-разрушительные «Bountiful Cores» (масштабируются по HP Нилу).

## Кратко

- **Стихия:** Гидро.
- **Оружие:** Меч.
- **Регион:** Сумеру.
- **Релиз:** 3.1.

## Роль

- **Bloom-enabler** через A1 (требует моно-гидро/дендро).
- **Sub-DPS** через E (танцевальная серия).
- **HP-scaling** через все таланты.

## Таланты

| Талант | Назначение |
|--------|------------|
| NA: Dance of Samser | Меч-комба |
| E: Dance of Haftkarsvar | 3 гидро-удара по AoE |
| Q: Dance of Abzendegi: Distant Dreams | Гидро-AoE, тегает противников |

## Созвездия

| C | Эффект |
|:-:|--------|
| C1 | Bountiful Core +60% урон |
| C2 | Core +30% AoE |
| C6 | Bountiful Core генерирует HP-bonus команде |

## Билд

- **Сет:** Gilded Dreams (4) или Deepwood Memories (4).
- **Часы:** HP%, **кубок:** HP%, **шапка:** HP%.
- **ER:** 130–160%.

## Команды

- Bloom-моно: Нилу + Накида + Бай Чжу + Кокоми.
- Бюджет Bloom-моно: Нилу + Накида + Барбара + Яо Яо.
- Hyperbloom-вариант: Нилу + Накида + Сюй Цю + Куки.

- Требует моно-гидро/дендро (нет других стихий!).
