Пользовательские гайды — будущий раздел для материалов от комьюнити: команды, билды, маршруты фарма, прохождения, лор-теории и разборы патчей. Пока форма загрузки не встроена в сайт, эта страница фиксирует редакционный стандарт для авторов.

## Что можно отправлять

| Тип материала | Пример |
|---------------|--------|
| Билд персонажа | «Куки Шинобу Hyperbloom: бюджетная сборка 4★» |
| Командный разбор | «Аяка Freeze без Шэнь Хэ: варианты для F2P» |
| Маршрут фарма | «168 Qingxin за 3 захода» |
| Прохождение | «Как закрыть квестовую цепочку Aranyaka без спойлеров» |
| Лор-теория | «Почему Descenders связаны с Каэнри'а» |
| Экономика | «Как накопить 180 круток за 4 патча» |

## Формат публикации

Черновик пишется в Markdown и должен содержать frontmatter:

```yaml
---
title: "Название гайда"
section: community/user-guides
slug: "short-readable-slug"
status: draft
summary: "Краткое описание в 1-2 предложения."
gameVersion: "6.x"
updatedAt: 2026-05-13
author: "nickname"
sources:
  web:
    - {url: "https://example.com", title: "Источник", accessed: "2026-05-13"}
---
```

## Процесс публикации

1. Автор готовит черновик в Markdown.
2. Редакция проверяет факты, патч-актуальность, тон и источники.
3. При необходимости автор получает правки.
4. После одобрения материал публикуется в `/community/user-guides/<slug>`.
5. Лучшие материалы могут попасть в раздел [Лучшие гайды](/community/best-guides).

## Стандарты

- Писать своими словами, без копипаста из Wiki, KQM, Game8 и HoYoLAB.
- Указывать источники для чисел, дат, механик и таблиц.
- Спойлеры по сюжетным квестам выносить в отдельный блок.
- Не использовать «имба/мусор» без объяснения условий.
- Для билдов указывать: роль, оружие, артефакты, статы, ER, команды, ограничения.
- Для гайдов по квестам не ломать прохождение: шаги должны быть проверяемыми.

## Что не принимается

- Продажа аккаунтов, накрутка примогемов, сторонние магазины.
- Утечки без явной пометки «не подтверждено».
- Автоматически сгенерированные тексты без редакторской вычитки.
- Дословные переводы сторонних гайдов без разрешения.
