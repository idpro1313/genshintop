# Инадзумская усадьба: Изысканное поместье

Инадзумская усадьба: Изысканное поместье 
 Инадзумская усадьба: Изысканное поместье 
 Инадзумская усадьба: Изысканное поместье — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Изящный особняк в инадзумском стиле, созданный для Чайника безмятежности по проекту «Шёлкового принта». Инадзумская архитектура совершенствовалась на протяжении веков, по мере необходимости вбирая в себя лучшие образцы зарубежной архитектуры. В этом доме сочетаются элегантность усадьбы высшего класса с простотой народной архитектуры. Его планировка обеспечивает простор внутренних покоев. Внутреннее убранство элегантно, но при этом лишено пафоса - дом невероятно уютен, словно его выстроили для путешественницы и Паймон добрые друзья-инадзумцы.
