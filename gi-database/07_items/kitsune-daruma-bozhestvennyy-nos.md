# Кицунэ дарума «Божественный нос»

Кицунэ дарума «Божественный нос» 
 Кицунэ дарума «Божественный нос» 
 Кицунэ дарума «Божественный нос» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Большая кукла в форме лисы, которую невозможно опрокинуть, основана на персонаже из популярного романа издательского дома Яэ: шеф-повар, в котором течёт кровь благородной пурпурной лисы, обладает «божественным обонянием» и может в точности определить состояние и количество ингредиентов по запаху, что позволяет ему готовить с большой точностью. Эта кукла занимает шестое место по объёмам продаж всех сопутствующих товаров к романам издательства Яэ, иногда её продают на фестивалях. По словам покупателей, когда они смотрят на куклу во время еды, то могут вспомнить вкусные описания еды из романа, которые сильно возбуждают аппетит.
