# Звёздчатый гибискус: Осенний мандарин

Звёздчатый гибискус: Осенний мандарин 
 Звёздчатый гибискус: Осенний мандарин 
 Звёздчатый гибискус: Осенний мандарин — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Звёздчатый гибискус цвета осеннего мандарина. Он вызывает в памяти образы ветвей, усыпанных плодами, но, увы, сам он несъедобен.
