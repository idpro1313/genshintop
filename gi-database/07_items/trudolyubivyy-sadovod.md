# Трудолюбивый садовод

Трудолюбивый садовод 
 Трудолюбивый садовод 
 Трудолюбивый садовод — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Шпалера из дерева отоги, с помощью которой можно выращивать разные виды фруктов и овощей. Хотя сёгунат каждый месяц выделяет субсидии фермерам для поддержки местного садоводства, но никто не знает, сколько именно моры достаётся фермерам. А потому некоторые подумывают о том, чтобы начать выращивать растения, которые пользуются большим спросом, чем фрукты.
