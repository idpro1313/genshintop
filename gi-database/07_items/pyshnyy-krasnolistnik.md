# Пышный краснолистник

Пышный краснолистник 
 Пышный краснолистник 
 Пышный краснолистник — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Пышный краснолистник можно приобрести у торговцев. 
 Высокое пряморастущее дерево с роскошными листьями и ветвями, словно наполненными энергией самой земли. За ярко-красные листья его называют «краснолистник».
