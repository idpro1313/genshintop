# Нефритовое лисье ухо

Нефритовое лисье ухо 
 Нефритовое лисье ухо 
 Нефритовое лисье ухо — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Яркий главный цветок Инадзумы. Его тонкие, заострённые лепестки нежно-оранжевого цвета по форме напоминают лисьи уши.
