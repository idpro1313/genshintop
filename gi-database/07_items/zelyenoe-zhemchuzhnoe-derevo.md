# Зелёное жемчужное дерево

Зелёное жемчужное дерево 
 Зелёное жемчужное дерево 
 Зелёное жемчужное дерево — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Зелёное жемчужное дерево можно приобрести у торговцев. 
 За исключением нескольких более известных и ценных деревьев, жители Ли Юэ обычно называют деревья по их форме, вне зависимости от их видовой принадлежности. Например, дерево со светлым стволом называют «жемчужным». Для обрезки ветвей этого дерева не требуется острого лезвия или топора, поэтому многие искатели приключений используют его для изготовления различных базовых инструментов.
