 Лагерная стойка для луков: Резонансные перья 
 Лагерная стойка для луков: Резонансные перья — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Стойка для длинных луков распространённого в армии сёгуната образца. Под неброской внешностью этих луков скрываются превосходные боевые качества, заложенные в них мастерами острова Наруками. По объективным оценкам ремесленников из Фонтейна, эти луки выдерживают больший вес при натяжении и запуске, чем луки Сопротивления. У выпущенных из луков сёгуната стрел больше ударная мощь и скорость. Однако даже лучшим лучникам армии сёгуната, вооружённым этими шедеврами, не удалось справиться с «остроухим генералом» повстанцев несмотря на перевесе сил в семеро против одного. Чтобы уцелеть, воинам пришлось обратиться к Кудзё Саре.
