# Простой деревянный забор

Простой деревянный забор 
 Простой деревянный забор 
 Простой деревянный забор — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Забор, который может преградить путь практически любому крупному зверю. Прост в установке. Он сделан из довольно лёгкого дерева, которое можно вырвать из земли и использовать в качестве дубины при возникновении опасности.
