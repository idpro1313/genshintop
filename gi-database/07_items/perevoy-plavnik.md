# Перьевой плавник

Перьевой плавник 
 Перьевой плавник 
 Перьевой плавник — обычный материал, основное применение которого это возвышение оружия. Помимо этого он также может быть использован с другой целью, а именно изготовление предметов Лунный плавник посредством алхимии. Перьевой плавник можно получить в качестве лута с монстров. 
 Фрагмент плавника, извлечённый из спины поверженного зверя Сюаньвэнь. Хотя Сюаньвэнь не может летать на большие расстояния, благодаря этим плавникам, напоминающим перья, он может быстро преследовать добычу.
