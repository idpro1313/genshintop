# Наживка из корундина

Наживка из корундина 
 Наживка из корундина 
 Наживка из корундина — один из видов наживки для рыбной ловли. Наживка из корундина можно создать при помощи алхимии. 
 Удивительная натланская наживка из смеси плода корундина и злакофрукта. Она долго держит форму в воде, постепенно выделяя аромат и непрерывно привлекая рыбу. Подходит для ловли обычной рыбы-бойца, рыбы-солнца и обычной рыбы-единорога.
