# Изумрудные поля: Котик

Изумрудные поля: Котик-официант 
 Изумрудные поля: Котик-официант 
 Изумрудные поля: Котик-официант — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Изумрудные поля: Котик-официант можно приобрести у торговцев. 
 Дорожный знак в саду «Изумрудные поля». Он сделан из куста в образе кота, который похож на почтительного официанта. «Изумрудные поля» черпают вдохновение из сказочных стран чудес. Такие коты-официанты вручают детям приглашение по просьбе королевы Страны чудес в ключевые моменты истории. Обычно эти дорожные знаки размещают вблизи важных достопримечательностей.
