# Гортензия: Шёлковый стручок

Гортензия: Шёлковый стручок 
 Гортензия: Шёлковый стручок 
 Гортензия: Шёлковый стручок — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Шёлковый стручок, цвет которого напоминает первые лучи рассвета. Период его цветения длится дольше, чем восход солнца, поэтому у вас будет достаточно времени, чтобы насладиться его цветами.
