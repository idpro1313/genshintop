# Бесчисленные дома освещают луну

 «Бесчисленные дома освещают луну» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Подробная схема изготовления предмета декора «Бесчисленные дома освещают луну». После использования чертежа вы можете перейти в меню создания для просмотра.
