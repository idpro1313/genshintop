# Ступенчатый камень

Ступенчатый камень 
 Ступенчатый камень 
 Ступенчатый камень — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Ступенчатый камень можно приобрести у торговцев. 
 Считается, что много лет назад через горы протекала стремительная река, создавая препятствия для первых ступивших туда людей. Они брали камни и клали их в воду, чтобы пересечь реку. С тех пор плоские камни называют «ступенчатыми».
