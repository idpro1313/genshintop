# Огонёк ремонтного мека

Огонёк ремонтного мека 
 Огонёк ремонтного мека 
 Огонёк ремонтного мека — один из видов наживки для рыбной ловли. Огонёк ремонтного мека можно создать при помощи алхимии. 
 Уникальная наживка, в которой, как утверждается, используется технология, разработанная Исследовательским институтом Фонтейна. Устройство оснащено металлической защитной клеткой и крюкообразными прутьями, в него встроен светильник, изготовленный из экстракта маркота. При погружении в воду светильник мигает с определённой частотой, привлекая находящихся поблизости ремонтных меков, которых часто называют «подводными чистильщиками» или железными рыбами. Ремонтные меки могут сильно укусить, но только в пределах специально изготовленных внешних стержней, которые прочно удерживают их на месте. По сути, это специальная приманка, предназначенная для ловли ценных «железных рыбок». Подходит для ловли ремонтных меков.
