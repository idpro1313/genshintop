# Рыбацкий ремонт

Рыбацкий ремонт 
 Рыбацкий ремонт 
 Рыбацкий ремонт — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Простой деревянный сарай. Его основная конструкция сделана из древесины адхигамы и сияющей древесины. Такие сараи обычно строят из того, что осталось после возведения больших зданий или из пригодных к использованию остатков ветхих домов, чтобы посильнее сэкономить. В качестве материала для покрытия также часто используется выброшенный ковёр. В большинстве случаев такие сараи служат только для временного хранения товаров и ящиков, ожидающих доставки.
