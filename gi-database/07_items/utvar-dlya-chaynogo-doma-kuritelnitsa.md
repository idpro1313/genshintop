# Утварь для чайного дома: Курительница

 Утварь для чайного дома: Курительница 
 Утварь для чайного дома: Курительница — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Отменный сосуд из клёна, частый атрибут набора для чайной церемонии, в котором можно подать щедрое количество сладостей и закусок. Коробка для закусок внутри него состоит из двух отделений. Основание у коробки весьма тонкое, и в нижнее отделение можно положить лёд или горячий песок, чтобы поддерживать оптимальную температуру лакомств.
