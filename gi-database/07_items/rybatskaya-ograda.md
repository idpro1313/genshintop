# Рыбацкая ограда

Рыбацкая ограда 
 Рыбацкая ограда 
 Рыбацкая ограда — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Простая изгородь из сияющей древесины и лозы, украшенная узором из перевёрнутых треугольников. Говорят, что раньше лесные стражи устанавливали такие ограждения вокруг зон, где запрещено ловить рыбу, чтобы поддерживать экологический баланс. Эти области открывались для рыбаков только в определённые периоды. Однако с уменьшением активности рыболовов такие ограждения стали использоваться гораздо реже. Большинство людей используют эти ограждения только вблизи своих деревень.
