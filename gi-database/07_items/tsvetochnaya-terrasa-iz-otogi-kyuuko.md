# Цветочная терраса из отоги «Кюуко»

Цветочная терраса из отоги «Кюуко» 
 Цветочная терраса из отоги «Кюуко» 
 Цветочная терраса из отоги «Кюуко» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Трёхъярусная подставка-лесенка для цветов. Тип и плотность посадки цветов в каждом слое, способ подрезки корневой системы, толщина используемой почвы, степень уплотнения и влажности - всё это строго регламентировано. Только настоящие мастера икебаны могут точно выполнить все эти требования, чтобы цветы в клумбе надолго сохранили свой цвет и аромат.
