 «Секрет охотника» 
 «Секрет охотника» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). «Секрет охотника» можно приобрести у торговцев. 
 Довольно высокое растение с большим количеством стеблей, которые можно использовать в качестве опор для лазания. Охотники-ветераны из Мондштадта прячутся в его зелени, чтобы следить за своей добычей.
