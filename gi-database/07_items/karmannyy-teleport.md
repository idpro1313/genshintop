# Карманный телепорт

Карманный телепорт 
 Карманный телепорт 
 Карманный телепорт — крайне полезный инструмент позволяющий создать в любом месте точку для телепортации. К сожалению точка существует лишь 7 дней и для создания новой вам придется создать еще один предмет. Карманный телепорт можно создать при помощи алхимии. 
 Используйте этот предмет, чтобы создать точку телепортации в месте применения. Одновременно может существовать только одна такая точка. Карманный инструмент для создания временных точек телепортации. Он соединяется с артериями земли на уровне, достаточном для исполнения телепортации. Очень полезный инструмент для обладателей Глаза Бога. Тем не менее механизм работы точек телепортации ещё не до конца изучен, поэтому то, к каким последствиям может привести их использование, никому неизвестно...
