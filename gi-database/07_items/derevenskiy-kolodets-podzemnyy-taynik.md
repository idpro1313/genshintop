# Деревенский колодец: Подземный тайник

 Деревенский колодец: Подземный тайник 
 Деревенский колодец: Подземный тайник — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Сухой колодец, который иногда можно увидеть в инадзумских деревнях. Источник воды давно иссяк, но внизу остаётся глубокое большое пространство, опасное даже для взрослых. Если заигравшийся ребёнок упадёт вниз... последствия будут весьма плачевными. Поэтому такие сухие колодцы чаще всего закрывают крышками. Однако иногда их используют для сокрытия ужасных тайн...
