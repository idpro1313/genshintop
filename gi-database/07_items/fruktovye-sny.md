# Фруктовые сны

Фруктовые сны 
 Фруктовые сны 
 Фруктовые сны — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Декоративные воздушные шары, изготовленные из порхающих помидорок. Прочные верёвки сделаны из волокон, добытых из стеблей порхающих помидорок. Они слишком хотят взмыть в небо, поэтому верёвки необходимы, чтобы удержать их на земле.
