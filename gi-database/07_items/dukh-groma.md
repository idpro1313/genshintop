# Дух грома

Дух грома 
 Дух грома 
 Дух грома — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Модель инадзумского пиратского корабля. Она великолепно передаёт образ судна, рассекающего прибой. Говорят, если присмотреться, на корпусе корабля можно увидеть следующие слова: «Во-вторых, моряки никогда не прощаются вслух».
