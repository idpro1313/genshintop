# Бочка из отоги: Утоление жажды

Бочка из отоги: Утоление жажды 
 Бочка из отоги: Утоление жажды 
 Бочка из отоги: Утоление жажды — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Деревянная бочка в деревенском стиле, оснащена черпаком для воды. Деревенские жители часто дочиста отмывают бочки, чтобы воду или вино можно было пить прямо из них. Когда у жителей деревни спрашивают, чистая ли вода в бочке, они отвечают: «Пить можно».
