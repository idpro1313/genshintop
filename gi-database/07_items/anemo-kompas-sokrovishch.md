# Анемо компас сокровищ

Анемо компас сокровищ 
 Анемо компас сокровищ 
 Анемо компас сокровищ — инструмент, при использовании которого отображает направление ближайшего к персонажу сундука. Может быть использован только в локации Мондштадт и не тратится при использовании. Рецепт для его создания открывается за 6 уровень репутации. Анемо компас сокровищ можно создать с помощью кузнеца. 
 При использовании в Мондштадте указывает на сундуки поблизости. Стрелка компаса следует не за мировым магнитным полем, а за энергией артерий земли, которые ведут к сокровищам.
