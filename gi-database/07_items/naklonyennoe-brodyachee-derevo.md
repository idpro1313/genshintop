# Наклонённое бродячее дерево

 Наклонённое бродячее дерево 
 Наклонённое бродячее дерево — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Наклонённое бродячее дерево можно приобрести у торговцев. 
 Очень распространённое дерево в Сумеру. Его называют бродячим деревом из-за его широкого географического присутствия в Сумеру. Когда оно растёт в рыхлой почве или рядом с горой, его ствол естественным образом наклоняется.
