# Сущность потухшей звезды

Сущность потухшей звезды 
 Сущность потухшей звезды 
 Сущность потухшей звезды — один из материалов событий. Способы получения: Испытание «Упавшая звезда». 
 Очищенная сущность звёздного тела. Можно обменять на сокровища во время события «Заблудшие звёзды». Эссенция далёкой звезды. Содержащаяся в ней сила излучает уютное тепло.
