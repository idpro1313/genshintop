# Изгородь из отоги «Кинтакэ»

 Изгородь из отоги «Кинтакэ» 
 Изгородь из отоги «Кинтакэ» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Ограда из отоги, которая годами не знала починки. Платой за получение «почти настоящего бамбука» с помощью техники «кинтакэ» является сложность в подгонке досок, в результате между ними остаются щели, через которые во дворе могут проникнуть небольшие животные и доставить жильцам немало хлопот.
