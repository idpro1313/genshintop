Катализатор пламени 
 Катализатор пламени 
 Катализатор пламени — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Катализатор пламени можно приобрести у торговцев. 
 Редкий цветок цвета пламени с лепестками странной формы. По результатам экспериментов учёных, этот цветок содержит особое каталитическое вещество, которое может заставить прозрачные экстракты из люминесцентных растений излучать мощный свет, будто они достигли стадии зрелости.
