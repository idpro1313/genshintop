# Фонарь из белого камня «Мэйчу»

 Фонарь из белого камня «Мэйчу» 
 Фонарь из белого камня «Мэйчу» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Искусно спроектированный уличный фонарь Инадзумы, сделанный из чистого белого камня. Говорят, что раскаты грома остаются в них, демонстрируя, таким образом, мощь сёгуна Райдэн. Однако любопытные дети неоднократно проводили эксперименты, но так и не услышали громовое эхо.
