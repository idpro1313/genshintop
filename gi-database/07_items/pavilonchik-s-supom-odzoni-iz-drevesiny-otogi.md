# Павильончик с супом одзони из древесины отоги

Павильончик с супом одзони из древесины отоги 
 Павильончик с супом одзони из древесины отоги 
 Павильончик с супом одзони из древесины отоги — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Такой передвижной ларёк под названием «ятай» является комбинацией тележки и стола из древесины отоги. Такие ларьки в Инадзуме встречаются часто. Суп одзони - популярное в городе Инадзума блюдо, его без преувеличения можно назвать блюдом для всех. Поэтому в Инадзуме так много тележек, которые им торгуют. Некоторые владельцы предлагают покупателям самим добавить ингредиенты в кипящий суп, чтобы не только утолить голод, но и дать волю вдохновению.
