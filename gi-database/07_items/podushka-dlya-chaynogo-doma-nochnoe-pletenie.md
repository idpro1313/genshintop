# Подушка для чайного дома: Ночное плетение

Подушка для чайного дома: Ночное плетение 
 Подушка для чайного дома: Ночное плетение 
 Подушка для чайного дома: Ночное плетение — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Подушка из чайного дома «Коморэ», сплетённая из сушёной соломы. Способ плетения чрезвычайно замысловат, но в использовании эта подушка ничем не отличается от любой другой. «Коморэ» изобилует правилами, но они по большей части относятся к основным правилам приличия и гостеприимства. В чайном доме никого особо не заботит ни подбор материалов, ни показная роскошь.
