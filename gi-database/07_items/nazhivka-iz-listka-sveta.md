Наживка из листка света 
 Наживка из листка света 
 Наживка из листка света — один из видов наживки для рыбной ловли. Наживка из листка света можно создать при помощи алхимии. 
 Удивительная натланская наживка из смеси цветов пепельника и пшеницы. Она поддерживает особую температуру под водой, привлекая чувствительных к теплу рыб. Подходит для ловли живущей во флогистоне рыбы.
