# Бутыль ловца ветра

Бутыль ловца ветра 
 Бутыль ловца ветра 
 Бутыль ловца ветра — уникальный инструмент позволяющий вашему персонажу взмыть в воздух. Он не тратится при использовании а чертеж для его создания открывается за третьем уровне репутации Мондштадта. Бутыль ловца ветра можно создать с помощью кузнеца. 
 Используется для хранения анемогранумов, с помощью которых можно создать ветряные потоки. Учёные Ордо Фавониус создали этот прибор, чтобы рыцари могли использовать милость Анемо Архонта в своих походах. Но, строго говоря, чтобы пользоваться планером, нужна лётная лицензия, что несколько ограничило применение бутыли (во всяком случае, законным образом).
