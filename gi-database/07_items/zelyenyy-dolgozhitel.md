# Зелёный долгожитель

Зелёный долгожитель 
 Зелёный долгожитель 
 Зелёный долгожитель — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Зелёный долгожитель можно приобрести у торговцев. 
 Очень толстое дерево с пышной листвой, способное обеспечить отличное укрытие. Его роскошная крона имеет особый оттенок насыщенного зелёного цвета, по слухам, этот цвет указывает на то, что оно живёт не менее тысячи лет. Жаль, что такие деревья можно найти только на острове Наруками - в ходе активных боевых действий на Татарасуне и Ясиори огромное количество таких деревьев было уничтожено, ведь их густые кроны идеально подходили для засад вражеских лучников. И обе стороны поторопились вырубить деревья, чтобы устранить угрозу.
