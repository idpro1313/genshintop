 Волокно для фонарей 
 Волокно для фонарей один из материалов событий Основное применение этого материала это изготовление предметов Небесный фонарь посредством алхимии. Способы получения: Растения. 
 Используется для изготовления небесных фонарей во время Праздника морских фонарей. После особенной обработки из этого самого обыкновенного материала можно сделать покрытие для фонаря, которое позволит ему лететь далеко-далеко и нести с собой прекрасные мечты людей.
