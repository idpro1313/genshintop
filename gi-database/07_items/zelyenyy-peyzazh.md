# Зелёный пейзаж

Зелёный пейзаж 
 Зелёный пейзаж 
 Зелёный пейзаж — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Зелёный пейзаж можно приобрести у торговцев. 
 Поэты Ли Юэ называют дикие травы, которые можно увидеть повсюду, «зелёным пейзажем». Пышная зелёная растительность является символом приятного климата и устойчивости артерий земли, а также самым незаменимым типом ландшафта для процветающего мира.
