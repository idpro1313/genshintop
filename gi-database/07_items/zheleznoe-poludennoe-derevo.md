 Железное полуденное дерево — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Железное полуденное дерево можно приобрести у торговцев. 
 Высокое пышное дерево с зелёной кроной. Его древесина настолько прочна, что из неё изготавливают конструкции всевозможных зданий. Его даже используют в качестве основного материала для изготовления оружейных стволов. Поэтому оно получило название «железное». Это железное дерево с раскидистой зелёной кроной полностью достигло зрелости, и его можно использовать по назначению.
