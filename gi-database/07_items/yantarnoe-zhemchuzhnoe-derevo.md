# Янтарное жемчужное дерево

Янтарное жемчужное дерево 
 Янтарное жемчужное дерево 
 Янтарное жемчужное дерево — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Янтарное жемчужное дерево можно приобрести у торговцев. 
 За исключением нескольких более известных и ценных деревьев, жители Ли Юэ обычно называют деревья по их форме, вне зависимости от их видовой принадлежности. Например, дерево со светлым стволом называют «жемчужным». Полученную из янтарного жемчужного дерева густую целлюлозную массу можно использовать для изготовления большого количества бумаги. Большая часть дешевой бумаги на рынке изготовлена из этого дерева.
