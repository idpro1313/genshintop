# Торшер из клёна: Очищающее сияние

 Торшер из клёна: Очищающее сияние 
 Торшер из клёна: Очищающее сияние — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Крупный светильник из штаба комиссии Ясиро. Цена его высока, а для изготовления требуется немало материала и труда. Говорят, во времена экономического упадка комиссия Ясиро наняла ремесленников для изготовления большого количества продукции с тем, чтобы обеспечить им средства к существованию. Такие торшеры покупались как раз в тот период.
