Пух для фитиля 
 Пух для фитиля 
 Пух для фитиля один из материалов событий Основное применение этого материала это изготовление предметов Небесный фонарь посредством алхимии. Способы получения: Враги типа хиличурлов и Элитные Фатуи. 
 Используется для изготовления небесных фонарей во время Праздника морских фонарей. После особенной обработки из этого самого обыкновенного пуха можно сделать фитиль для фонаря, который позволит ему лететь далеко-далеко и нести с собой прекрасные мечты людей. 
 Применение в алхимии 
 Материал Пух для фитиля может быть применён в алхимии для создания следующих предметов
