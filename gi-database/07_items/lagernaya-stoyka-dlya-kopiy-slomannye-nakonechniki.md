 Лагерная стойка для копий: Сломанные наконечники 
 Лагерная стойка для копий: Сломанные наконечники — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Стойка для копий распространённого в армии сёгуната образца. Древки копий делают из аралии и обрабатывают огнеупорным составом, чтобы они не загорелись в пылу сражения. В отличие от крыш казармы копья сёгуната обработаны огнеупорным покрытием, которое, по слухам, было разработано много лет назад для борьбы с демонами. Однако на сегодняшний день никаких доказательств этого не уцелело.
