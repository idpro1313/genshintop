# Тысяча лепестков красной глазури

 Тысяча лепестков красной глазури — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Тысяча лепестков красной глазури можно приобрести у торговцев. 
 Благодаря шаровидным цветам пестрого окраса этот вид цветов культивируется как декоративное растение. По пушистости они не уступают мондштадтским одуванчикам. Лёгкое дуновение ветерка уносит яркие лепестки в воздух, где они порхают в медленном танце.
