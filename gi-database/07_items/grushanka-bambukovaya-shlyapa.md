# Грушанка: Бамбуковая шляпа

Грушанка: Бамбуковая шляпа 
 Грушанка: Бамбуковая шляпа 
 Грушанка: Бамбуковая шляпа — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Светло-зелёная грушанка, по форме напоминает бамбуковую шляпу. По мере роста её цвет меняется с изумрудного на зелёный, а затем на светло-жёлтый. Однако зеленоватый цвет на обратной стороне лепестков никогда не пропадает, как будто на нём всегда надета зеленоватая бамбуковая шляпка. Отсюда и название этого цветка.
