 Рыбацкое консервирование 
 Рыбацкое консервирование — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Сушилка для рыбы из древесины адхигамы, покрытая сверху пластинами из сияющей древесины. Она проста в изготовлении и хранении, по слухам, без неё не обходится ни одна рыбацкая семья. Правильно высушенная рыба может храниться в течение длительного времени, не теряя своего особого вкуса. Любители сушёной рыбы едят её с сумерскими специями, чтобы добавить немного аромата.
