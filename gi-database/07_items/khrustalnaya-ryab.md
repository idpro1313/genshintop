# Хрустальная рябь

Хрустальная рябь 
 Хрустальная рябь 
 Хрустальная рябь — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Основа, повторяющая пруды в особняках Ли Юэ. Кристально чистая вода сделает любой помещённый в неё цветок ещё более прекрасным, а затем отразит его красоту в ряби по всей поверхности пруда.
