# «Спасение добычи»

«Спасение добычи» 
 «Спасение добычи» 
 «Спасение добычи» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). «Спасение добычи» можно приобрести у торговцев. 
 Растение с множеством ветвей и пышными листьями. В процессе борьбы с охотниками некоторые умные существа научились прятаться в кустах, надевая их как камуфляжную мантию. Такая изобретательность - большой недостаток для начинающих охотников.
