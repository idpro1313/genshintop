# Изумрудные поля: Дорожка из печенья

Изумрудные поля: Дорожка из печенья 
 Изумрудные поля: Дорожка из печенья 
 Изумрудные поля: Дорожка из печенья — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Каменная дорожка сада «Изумрудные поля». Она сделана из чистых, ярких и крепких камней. Её можно использовать для мощения двора. «Изумрудные поля» черпают вдохновение из сказочных стран чудес. В некоторых сказках описываются духи, которые едят камни, поэтому детям нравятся каменные дорожки, они называют их «печеньем для фей».
