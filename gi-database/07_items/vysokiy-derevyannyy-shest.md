# Высокий деревянный шест

Высокий деревянный шест 
 Высокий деревянный шест 
 Высокий деревянный шест — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Базовая конструкция для строительства деревянного сарая. Иногда используется в качестве заметного маркера, указывающего на ключевое местоположение постройки.
