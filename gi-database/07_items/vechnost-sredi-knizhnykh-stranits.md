# Вечность среди книжных страниц

Вечность среди книжных страниц 
 Вечность среди книжных страниц 
 Вечность среди книжных страниц — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Утончённая книжная полка из берёзы. На ней могут храниться не только многочисленные книги. Она идеально подойдёт для демонстрации изысканных часов из Фонтейна. Часовые стрелки и горы книг олицетворяют пролетевшее время и бесконечное количество вещей, которые необходимо сделать.
