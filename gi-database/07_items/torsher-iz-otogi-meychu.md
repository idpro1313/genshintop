# Фонарь из отоги «Мэйчу»

 Фонарь из отоги «Мэйчу» 
 Фонарь из отоги «Мэйчу» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Искусно спроектированный уличный фонарь Инадзумы, сделанный из резного дерева отоги. Говорят, что раскаты грома остаются в них, демонстрируя, таким образом, мощь сёгуна Райдэн. Но, по словам стариков, эта легенда восходит к оригинальному мастеру, создавшему эти фонари, который выдумал бессмысленное название в порыве вдохновения, объединив «вечность Наруками» и «вечные молнии».
