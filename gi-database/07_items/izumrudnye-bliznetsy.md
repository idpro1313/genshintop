# Изумрудные близнецы

 Изумрудные близнецы 
 Изумрудные близнецы — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Изумрудные близнецы можно приобрести у торговцев. 
 Двойные сросшиеся вместе кусты. За листья прекрасного зелёного цвета их называют «изумрудными близнецами».
