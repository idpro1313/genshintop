# Тануки дарума «Подлинная кулинария»

Тануки дарума «Подлинная кулинария» 
 Тануки дарума «Подлинная кулинария» 
 Тануки дарума «Подлинная кулинария» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Большая кукла в форме тануки, которую невозможно опрокинуть, основана на персонаже из популярного романа издательского дома Яэ: шеф-повар случайно обрёл божественную силу и способен создать барьер «настоящей кулинарии», находясь за которым, другие повара лишаются своих дешёвых трюков и отныне могут пользоваться только своими реальными навыками. Благодаря этому персонаж выигрывает в кулинарном конкурсе. Эта кукла занимает двадцать второе место по объёмам продаж всех сопутствующих товаров к романам издательства Яэ, иногда её продают на фестивалях. По логике, как товар, связанный с главным персонажем романа, он должен хорошо продаваться. Однако после расследования выяснилось, что большинство читателей, которым нравится этот персонаж, - повара с весьма посредственными навыками. Возможно, у этого персонажа не та целевая аудитория...
