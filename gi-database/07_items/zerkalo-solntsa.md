# Зеркало солнца

Зеркало солнца 
 Зеркало солнца 
 Зеркало солнца — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Главный цветок, отражающий солнечный свет, словно зеркало. Этот свет способен пробить даже зловещие грозовые тучи, а в его лепестках чувствуется томление полуденного солнца.
