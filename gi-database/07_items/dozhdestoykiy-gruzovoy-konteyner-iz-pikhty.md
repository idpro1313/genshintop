# Дождестойкий грузовой контейнер из пихты

Дождестойкий грузовой контейнер из пихты 
 Дождестойкий грузовой контейнер из пихты 
 Дождестойкий грузовой контейнер из пихты — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Прочный пихтовый контейнер, обтянутый водонепроницаемой тканью, что обеспечивает сухость внутри. Он имеет большую вместимость и может хранить объёмные грузы, хотя за это приходится платить шокирующим весом, ведь для его транспортировки потребуется несколько человек. Говорят, что некоторые контейнеры можно запереть изнутри. Работникам в пути очень удобно спрятаться внутри и спастись в случае нападения монстров.
