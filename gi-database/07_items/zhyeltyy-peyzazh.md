# Жёлтый пейзаж

Жёлтый пейзаж 
 Жёлтый пейзаж 
 Жёлтый пейзаж — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Жёлтый пейзаж можно приобрести у торговцев. 
 Поэты Ли Юэ называют дикие травы, которые можно увидеть повсюду, «жёлтым пейзажем». Жёлтый цвет предвещает наступление холодов.
