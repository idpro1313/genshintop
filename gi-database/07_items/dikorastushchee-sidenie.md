# «Дикорастущее сидение»

«Дикорастущее сидение» 
 «Дикорастущее сидение» 
 «Дикорастущее сидение» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). «Дикорастущее сидение» можно приобрести у торговцев. 
 Круглый кустарник с мягкими ветвями и листьями, который можно легко выкорчевать. Из двух таких кустов получится хорошее сидение.
