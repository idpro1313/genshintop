# Военный барабан: Звук победы

 Военный барабан: Звук победы 
 Военный барабан: Звук победы — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Боевой барабан, часто встречающийся в военных лагерях, выполнен в традиционном стиле, который передаётся в Инадзуме на протяжении веков. На поверхности барабана изображена эмблема Сангономии, но звук, который он издаёт, идентичен звукам барабанов, используемых сёгунатом. Даже опытные барабанщики не заметят разницы. Среди множества тактик армии сопротивления Сангономии есть тактика «использования идентичной барабанной дроби, чтобы помешать силам сёгуната». Хотя она не нашла широкого применения, всё же она используется для перелома хода битвы в критический момент.
