# Фальшивый червь

Фальшивый червь 
 Фальшивый червь 
 Фальшивый червь — один из видов наживки для рыбной ловли. Фальшивый червь можно создать при помощи алхимии. 
 Эта имитирующая червя приманка сделана в основном из конденсата. Влекомая подводными течениями, она активно движется в воде, что делает её смертельно привлекательной для слишком любопытных рыб. Профессионалы также добавляют немного фруктов, чтобы усилить аромат и сделать её совершенно неотразимой. Подходит для ловли рыб-бабочек.
