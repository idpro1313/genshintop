# Деревянная бочка для перевозки грунта

Деревянная бочка для перевозки грунта 
 Деревянная бочка для перевозки грунта 
 Деревянная бочка для перевозки грунта — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Деревянная кадка, наполненная землёй. При разработке шахт и строительстве домов часто скапливается много земли, которая остаётся неиспользованной. Иногда нет возможности подвезти тележку к месту строительства, поэтому для транспортировки грунта используются такие кадки. Некоторые предприимчивые работники выращивают цветы и растения прямо в них.
