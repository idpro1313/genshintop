# Гардероб из юмэмиру «Росинка»

 Гардероб из юмэмиру «Росинка» 
 Гардероб из юмэмиру «Росинка» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Закрытый инадзумский гардероб. Его толщина и стенки были скорректированы с учетом уникальных особенностей кимоно жителей Инадзумы. Инадзума с трёх сторон окружена океаном, а воздух очень влажный, поэтому одежда может легко промокнуть и заплесневеть во время длительного хранения. Поэтому верхняя секция этого шкафа содержит сменную деревянную коробку, наполненную высушенными лепестками и опилками, которые используются для поглощения воды и контроля влажности внутри. И кстати, получается довольно хорошо.
