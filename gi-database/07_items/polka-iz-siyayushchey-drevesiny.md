 Полка из сияющей древесины 
 Полка из сияющей древесины — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Эта полка для хранения изготовлена из высококачественной сияющей древесины. Её конструкция проста, прочна и долговечна и подходит для хранения большого количества товаров. Мастера учли особую архитектуру некоторых сумерских зданий, где двери не имеют правильной прямоугольной формы, и добавили возможность быстро собирать и разбирать полку для удобства транспортировки.
