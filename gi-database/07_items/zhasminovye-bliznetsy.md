# Жасминовые близнецы

 Жасминовые близнецы 
 Жасминовые близнецы — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Жасминовые близнецы можно приобрести у торговцев. 
 Двойные сросшиеся вместе кусты. За уникальный цвет их называют «жасминовыми близнецами».
