# Тумбочка из песчаного дерева с золотым узором

Тумбочка из песчаного дерева с золотым узором 
 Тумбочка из песчаного дерева с золотым узором 
 Тумбочка из песчаного дерева с золотым узором — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Прикроватная тумбочка из песчаного дерева в стиле Ли Юэ. Её ножки обиты металлом, что делает её такой же прочной, как и красивой. В ящике достаточно места, чтобы вместить даже катализатор для самообороны, так что владелец может сразу начать контратаку в случае опасности. Говорят, что эта тумбочка стала популярной благодаря биографии Воли Небес.
