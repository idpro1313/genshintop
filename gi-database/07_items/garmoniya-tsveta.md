# Гармония цвета

Гармония цвета 
 Гармония цвета 
 Гармония цвета — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). Гармония цвета можно приобрести у торговцев. 
 Красивый оранжевый цветок, который иногда можно увидеть в дикой природе в Сумеру. Его также выращивают в лабораториях Пардис Дхяй. Помимо использования в качестве декоративного цветка, из его лепестков можно извлечь специальный краситель, который придаёт цвет люминесцентным материалам. Как правило, для интерьера помещений предпочтительны тёплые цвета, однако большинство люминесцентных материалов имеют холодный, белый цвет. Поэтому для исправления ситуации необходимо использовать эти цветы.
