# Ширма для горячего источника: Умеренность

Ширма для горячего источника: Умеренность 
 Ширма для горячего источника: Умеренность 
 Ширма для горячего источника: Умеренность — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Складная ширма из аралии. Такими ширмами можно отделять отдельные участки горячих источников. Люди могут оказаться излишне разгорячёнными, выпивая и любуясь луной в горячих источниках. По соображениям безопасности в воде пить запрещено, а также есть отдельная зона для дегустации вин.
