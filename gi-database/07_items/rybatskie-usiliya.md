# Рыбацкие усилия

Рыбацкие усилия 
 Рыбацкие усилия 
 Рыбацкие усилия — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Корзина, сплетённая из тонких и упругих полос сияющей древесины, а каркас изготовлен из древесины адхигамы. Говорят, что такая корзина «почти невесомая», и даже ребёнок может с лёгкостью её поднять. Поскольку корзина сделана из отдельных пластов древесины, между ними есть заметные щели. Если потрясти корзину, ухватившись за ручку, то можно вытряхнуть лишнюю воду и, следовательно, избавиться от лишнего веса.
