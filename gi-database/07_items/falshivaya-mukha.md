# Фальшивая муха

Фальшивая муха 
 Фальшивая муха 
 Фальшивая муха — один из видов наживки для рыбной ловли. Фальшивая муха можно создать при помощи алхимии. 
 Имитирующая муху нахлыстовая приманка, изготовленная из ярких материалов. Она не имеет запаха и особенно эффективна для привлечения крупных рыб с хорошим зрением. Подходит для самых продвинутых техник ловли. Гильдии рыболовов в некоторых регионах выдвигают умение пользоваться этой наживкой в качестве обязательного требования при повышении квалификации. Подходит для ловли лжедраконов, иглобрюхов и скатов.
