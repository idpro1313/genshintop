# Зонтик из сияющего камня

Зонтик из сияющего камня 
 Зонтик из сияющего камня 
 Зонтик из сияющего камня — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Очаровательное настольное украшение, формой смутно напоминающее гриб. Сделано из странного материала, похожего на камень. Среди чужеземцев на Рито ходят слухи, что материалом для этого украшения служил камень из леса Тиндзю. Будто бы если вскрыть камень, возраст которого насчитывает тысячу лет, то внутри обнаружится светящаяся голубым жидкость. На самом деле, мастер просто нанёс на обычную скульптуру фиолетовое покрытие из несъедобного желе.
