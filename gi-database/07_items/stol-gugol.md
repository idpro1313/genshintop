# Стол «Гугол»

Стол «Гугол» 
 Стол «Гугол» 
 Стол «Гугол» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Таинственный стол неизвестного происхождения. Сочетание цветов оставляет неизгладимое впечатление в памяти. Сложно сказать, есть ли в его конструкции какой-то секрет, позволяющий ему прекрасно служить как для приёма пищи, так и в качестве рабочего стола. Некоторые знающие мастера называют эту особенность «исключительной эргономикой».
