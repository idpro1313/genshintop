Ягодная наживка 
 Ягодная наживка 
 Ягодная наживка — один из видов наживки для рыбной ловли. Ягодная наживка можно создать при помощи алхимии. 
 Распространённая наживка в Нод-Крае, изготовленная из летники и муки. Хорошо привлекает обычную рыбу... Подходит для ловли обыкновенной рыбы-топора.
