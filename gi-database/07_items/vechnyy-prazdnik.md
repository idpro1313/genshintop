# «Вечный праздник»

«Вечный праздник» 
 «Вечный праздник» 
 «Вечный праздник» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). «Вечный праздник» можно приобрести у торговцев. 
 Довольно необычное растение с множеством больших, толстых листьев без запаха. После мытья листья можно использовать для приготовления пищи. Одного такого растения достаточно, чтобы на весь день обеспечить едой целый отряд искателей приключений.
