# Коралл из красного металла

Коралл из красного металла 
 Коралл из красного металла 
 Коралл из красного металла — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Яркое настольное украшение, которое могло быть создано только искусным резчиком. Говорят, что образцом для него стал выловленный из моря рыбаками коралл, и оригинал был настолько великолепен, что один богатый иностранный купец купил его за большие деньги.
