# Сельский чайный киоск

Сельский чайный киоск 
 Сельский чайный киоск 
 Сельский чайный киоск — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Киоск с различными чайными наборами. Каждый из них способен усилить вкус конкретного чая, благодаря чему этот ларек очень ценится его владельцами. Некоторые продавцы чая, помимо знания всех технологий приготовления этого напитка, умеют ещё и рассказывать истории, поэтому они владеют двумя профессиями одновременно.
