# Крепкая деревянная бочка

Крепкая деревянная бочка 
 Крепкая деревянная бочка 
 Крепкая деревянная бочка — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Эта крепкая деревянная кадка идёт в комплекте с верёвкой и двумя металлическими кольцами и имеет множество применений. Это одна из тех кадок, содержимое которых норовит разлиться на ноги несущего её человека.
