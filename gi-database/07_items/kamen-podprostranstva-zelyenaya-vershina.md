# Камень подпространства: Зелёная вершина

 Камень подпространства: Зелёная вершина 
 Камень подпространства: Зелёная вершина — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Одна из скал, составляющих часть основного рельефа обители Адепта. Выглядит самой устойчивой. Можно спросить мнение некоего Анемо Архонта о том, как превратить их в высокую башню. У него есть некоторый опыт избегания прискорбных конструкций.
