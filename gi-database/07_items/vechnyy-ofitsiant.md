# «Вечный официант»

«Вечный официант» 
 «Вечный официант» 
 «Вечный официант» — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). «Вечный официант» можно приобрести у торговцев. 
 Довольно необычное растение с множеством больших, толстых листьев без запаха. После мытья листья можно использовать для приготовления пищи. Более оптимистичные искатели приключений любят эти растения, сравнивая их с официантами ресторанов, которые разносят тарелки.
