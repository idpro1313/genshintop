 Подсвечник для чайного дома: Скрытый свет 
 Подсвечник для чайного дома: Скрытый свет — один из элементов декора, который можно разместить в Чайнике безмятежности (своей личной обители). 
 Лёгкий и изящный маленький светильник, который можно ставить где угодно. С тех пор как такие подсвечники стали использовать в чайном доме «Коморэ», они быстро приобрели широкую популярность. В его основании находится съёмная решетчатая пластина, которая позволяет легко заменить свечу. В нём даже есть особое крепление для свечи - она не выпадет, если с подсвечником побежать - и, разумеется, не загорится колпак. Поистине безопасно.
