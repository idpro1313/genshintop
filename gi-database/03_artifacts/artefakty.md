# Гайд по артефактам Genshin Impact

Общая информация об артефактах 
 Артефакты — это один из предметов экипировки позволяющий усилить вашего персонажа. Всего на одного персонажа можно надеть пять артефактов, каждый в свой слот: Корона разума, Кубок пространства, Пески времени, Перо смерти, Цветок жизни.
 Каждый артефакт может иметь от одного до двух бонусов комплекта, это означает что при надевании на персонажа двух артефактов из одного комплекта вы получите бонус, а при надевании четырех два бонуса.
