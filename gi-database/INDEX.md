# База данных Genshin Impact

Структурированная база знаний по игре Genshin Impact, собранная из открытых источников.

## Источники

- **genshindrop.io** - Русскоязычный сайт с гайдами
- **genshin-info.ru** - Русскоязычная вики по игре
- **add/** - Дополнительные гайды и прохождение
