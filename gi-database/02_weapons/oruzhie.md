# Гайд по оружию Genshin Impact

Общая информация об оружие 
 В игре Genshin Impact на данный момент присутствуют пять видов оружия: двуручные мечи, катализаторы, копья, луки и мечи. Каждый из персонажей игры может носить только определенный вид оружия.
 Помимо основного параметра(сила атаки), у данных предметов снаряжения есть дополнительная характеристика, а также дополнительный эффект. Именно основываясь на этих доп. параметрах необходимо подбирать оружие под героя.
