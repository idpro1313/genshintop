## Введение

Этот гайд создан специально для тех, кто только начинает играть в Genshin Impact. Вся информация здесь объяснена простым языком, без сложных игровых терминов.

---

## Основы управления

### Управление на ПК

| Кнопка | Действие |
|--------|----------|
| **W, A, S, D** | Ходьба/бег |
| **Пробел** | Прыжок |
| **ЛКМ** | Обычная атака |
| **ПКМ** | Заряженная атака |
| **E** | Элементальный навык |
| **Q** | Взрыв стихии (ультимейт) |
| **Shift** | Бег/спринт |
| **Левый Alt** | Смена персонажа |

### Управление на мобильных

- **Левый джойстик** — движение
- **Правый нижний угол** — атаки
- **Кнопки E и Q** — способности
- **Кнопки сбоку** — смена персонажа

## Способности персонажей

У каждого персонажа есть две основные способности:

1. **Элементальный навык (E)** — обычная способность, перезаряжается со временем. Не требует накопления энергии для использования.

2. **Взрыв стихии (Q)** — самая мощная способность персонажа. Для использования требуется накопить энергию.

**Как понять, какая способность важнее:**
- Читайте описание способностей
- Взрыв стихии обычно наносит больше урона или даёт важные бонусы
- Для некоторых персонажей основной урон — от обычных атак

---

## Энергия

Энергия нужна для использования взрыва стихии (кнопка Q). Это специальные частицы, которые вылетают из врагов, когда вы их бьёте.

**Как получить энергию:**

1. **Использование элементального навыка (E)** — основной источник
2. **Нанесение урона врагу** — некоторые враги отдают частицы при получении урона
3. **Убийство врагов**
4. **Совершение обычных атак** — даёт немного, но стабильно

**Совет:** Если ваш персонаж использует взрыв стихии часто, собирайте предметы с восстановлением энергии.

## Отряд

В отряде участвуют 4 персонажа. Вы можете менять их прямо во время боя.

**Как собрать отряд:**

1. Откройте меню (клавиша B или значок в углу)
2. Нажмите на портрет персонажа в отряде
3. Выберите другого персонажа

**Роли в отряде:**

- **Главный атакующий (Мейн ДД)** — персонаж, который наносит основной урон и находится на поле боя дольше всех
- **Второстепенный атакующий (Саб ДД)** — помогает наносить урон, но не основной
- **Поддержка (Саппорт)** — помогает отряду: лечит, даёт баффы, накладывает элементы

## Артефакты

Артефакты — это снаряжение, которое надевается на персонажей для усиления их характеристик.

### Типы артефактов

| Тип | Что даёт |
|-----|----------|
| **Цветок** | Только HP (здоровье) |
| **Перо** | Только сила атаки |
| **Часы** | Разные статы (можно выбрать) |
| **Кубок** | Разные статы или бонус урона стихии |
| **Шапка** | Разные статы или крит-шанс/урон |

### Сеты артефактов

Можно надевать 2 или 4 предмета из одного сета для получения бонуса.

**Пример:**
- Если надеть 2 части сета «Везунчик» — получите +100 защиты
- Если надеть 4 части — получите ещё и бонус лечения

**Как собирать сеты:**
- Наденьте 2 артефакта из одного сета
- Наденьте ещё 2 из другого сета
- Так можно получить два бонуса

### Какие статы важны для новичков

В начале игры сосредоточьтесь на:
- Сила атаки
- Восстановление энергии
- HP

Критические статы станут важны после 45 ранга приключений.

### Как улучшать артефакты

1. Выберите персонажа → Артефакты
2. Нажмите на артефакт
3. Нажмите «Улучшить»
4. Выберите другие артефакты для переплавки

**Важно:**
- Не удаляйте хорошие артефакты
- Используйте фильтры для поиска нужных
- Нажмите на замочек, чтобы случайно не удалить

## Валюты игры

### Мора

Основная валюта. Нужна для:
- Прокачки персонажей
- Улучшения артефактов
- Прокидки талантов
- Возвышения оружия

### Примогемы

Валюта для получения новых персонажей и оружия. Находится в инвентаре → Камни Истока.

### Смола

Ограниченный ресурс для прохождения подземелий и боёв с боссами.

## Баннеры и молитвы

Баннеры — это специальные окна, где можно «крутить» для получения новых персонажей и оружия.

### Типы баннеров

1. **Временный баннер (ивентый)** — содержит определённых персонажей с повышенным шансом выпадения
2. **Стандартный баннер** — содержит всех обычных персонажей и оружие

**Совет:** Всегда копите примогемы для временных баннеров, если хотите конкретного персонажа.

### Как крутить

1. Откройте меню молитв
2. Выберите баннер
3. Нажмите «Молитва» (1 раз) или «Молитва x10» (10 раз сразу)

### Гарантия

- Первый 5★ выпадает примерно на 70-90 попытке
- Если выпал не тот персонаж, следующий 5★ на временном баннере будет гарантированно тот, что на баннере
- Это называется «мягкий» и «жёсткий» гарант

## Созвездия

Созвездия — это дополнительные улучшения персонажа, которые активируются при получении дубликатов того же персонажа.

**Как активировать:**

1. Получите персонажа ещё раз (дубликат)
2. Откройте меню персонажа
3. Перейдите во вкладку «Созвездия»
4. Нажмите «Активировать»

**Важно:** После получения 6 созвездий (максимум) дальнейшие дубликаты дают звёздную пыль/блеск — валюту для магазина Паймон.

## Таланты

Таланты определяют, насколько сильно персонаж атакует и какие способности имеет.

### Типы талантов

- **Обычные атаки** — базовые удары
- **Элементальный навык (E)** — специальная способность
- **Взрыв стихии (Q)** — самая мощная способность

### Как прокачивать

1. Нажмите на персонажа → Таланты
2. Выберите нужный талант
3. Используйте книги талантов для повышения уровня

**Книги талантов бывают:**
- Зелёные (низкий уровень)
- Синие (средний уровень)
- Фиолетовые (высокий уровень)
- Золотые (максимальный уровень)

## Подземелья и дни недели

Подземелья — это специальные локации, где можно получить материалы. Они открыты не каждый день.

### Дни недели и доступные подземелья

| День | Подземелья |
|------|------------|
| Понедельник | Книги талантов (Сопротивление, Бездна) |
| Вторник | Книги талантов (Процветание, Золото) |
| Среда | Книги талантов (Сопротивление, Бездна) |
| Четверг | Книги талантов (Процветание, Золото) |
| Пятница | Книги талантов (Справедливость, Вечность) + Оружие |
| Суббота | Все типы книг + Оружие |
| Воскресенье | Любые подземелья |

## Ранг приключений

Ранг приключений (Adventure Rank, AR) — это ваш уровень в игре. Он определяет, какой контент вам доступен.

### Важные ранги

- **AR 20** — становятся доступны события
- **AR 30** — можно нормально фармить подземелья
- **AR 35-40** — основной этап прокачки
- **AR 45** — максимальный уровень артефактов (золотые)
- **AR 50** — можно полностью прокачать таланты и оружие

### Как повышать ранг

- Выполняйте задания
- Открывайте точки телепортации
- Активируйте статуи Архонтов
- Проходите подземелья
